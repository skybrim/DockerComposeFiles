# DockerComposeFiles
One-click deploy application by docker-compose.


debian/ubuntu 一键安装 docker 和 docker-compose
```shell
curl https://raw.githubusercontent.com/skybrim/ShellScript/master/debian_docker_install.sh | sh
```

docker-compose 常用命令
```
# 启动
docker-compose up -d
# 停止
docker-compose down
```
